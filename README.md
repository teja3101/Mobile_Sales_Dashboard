2. Description
This dashboard provides a comprehensive analysis of mobile sales performance by presenting key metrics such as total sales, quantity sold, number of transactions, and average sales values. It visualizes sales trends over time to help identify seasonal patterns and performance changes. The dashboard also highlights sales distribution across brands and cities, enabling geographic and product-level comparisons. Customer ratings are included to assess satisfaction and product performance. Additionally, insights into payment methods reveal customer purchasing behavior. Overall, this dashboard supports data-driven decision-making by offering clear and actionable sales insights.
